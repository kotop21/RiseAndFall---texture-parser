# Rise and Fall: Civilization at war Texture Converter

## Supported formats

* `.tga`
* `.dds`

## Folder structure

* `1_original` тАФ source textures
* `2_editing` тАФ extracted PNG files
* `3_packed` тАФ packed textures

## Commands

Extract textures:

```bash
converter unpack
```

Pack textures:

```bash
converter pack
```

Running the program without arguments creates the required folder structure automatically.
